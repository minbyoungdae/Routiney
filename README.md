        <p></p>
        데이터 저장을 위해 LocalStorage를 사용하였고, 협업 연습을 위해
        TypeScript를 활용했습니다. Text to speech는 Expo-speech를 활용하여
        구현하였으며, setInterval과 clearInterval를 사용하여 시작, 종료 기능을
        구현했습니다.
